Environmental Graphics

Project Overview

This environmental graphics project involved designing a comprehensive wayfinding and signage system for a modern office complex. The challenge was to create intuitive navigation while reinforcing the building's architectural identity.

The solution features a minimal, icon-based system with clear typography and strategic color coding. The design integrates seamlessly with the building's aesthetic while providing clear directional guidance and enhancing the overall user experience.

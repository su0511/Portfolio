2024 Editorial Design

Project Overview

A comprehensive editorial design project for a quarterly design magazine. The challenge was to create a layout system that would showcase diverse content while maintaining visual consistency and readability.

The solution involved developing a flexible grid system with carefully chosen typography that could accommodate various content types, from feature articles to image galleries.

Deliverables

  • Magazine layout design
  • Typography system
  • Grid structure guidelines
  • Cover design concepts
  • Print production files

Project Details

Client: Design Quarterly
Duration: 8 weeks
Role: Art Director
Tools: Adobe InDesign, Photoshop
